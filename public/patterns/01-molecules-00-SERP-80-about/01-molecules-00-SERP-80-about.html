About

This search result mock-up is a starting point for teams building enterprise search front ends, showing best practices. Over time I hope to store it in a Pattern Lab repository. The mock-up will then be part of a larger project with the working title «UX patterns for findability».

Findability is the key success factor of any search, and is the most important motivation for starting this project. To improve findability, the mock-up has an emphasis on:

  • Disambiguation through 1) Result groups and 2) Autocomplete topics
  • User influenced relevancy & findability
  • User added results & sources
  • Editorial maintenance
  • Findability reporting

For more on these topics, see the introduction on GitHub. Also see the inline comments for more details. (hide commentsComments are now hidden. Reload to show.)

Hopefully the rationale will be fleshed out in an accompanying presentation.

Content is in part sampled from NRK and The Guardian. CSS framework: Bootstrap 3. The mock-up is intended to be hosted on Github, but that hasn't happened yet.
